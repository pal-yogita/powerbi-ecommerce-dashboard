# Fire Analysis Dashboard

# Overview
This Power BI dashboard analyzes fire incidents, response times, and safety trends.  
It provides insights into:
- Major fire causes  
- Most affected areas  
- Emergency response times  
- Yearly and monthly fire trends  

# Objective
To help identify fire-prone areas and improve emergency preparedness.

#Tools Used
- Power BI  
- Excel/CSV data sources
